# Contributing

Bug reports and pull requests are welcome. Please open an issue first to discuss significant changes.

## Reporting Bugs

Open a [GitHub Issue](../../issues) with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your OS version and relevant versions (e.g. `whisper-cli --version`, `swift --version`)

## Submitting Changes

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Open a pull request with a clear description of what changed and why

No formal code style enforced — just keep it consistent with the surrounding code.
